ช่วยทำเวปแอพพลิเคชั่นในการวิเคราะห์และแสดงผลเปรียบเทียบผลลัพธ์ของงานวิจัย ทั้งเป็นข้อมูลตัวเลขและกราฟ
ตัวอย่างเนื้อหาข้อมูลทั้งหมดอยู่ในโฟร์เดอร์ ผลวิจัย
สามารถอัพโหลดชุดข้อมูลเพื่อเปรียบเทียบผลวิจัยได้
ออกแบบตามความเหมาะสมเพื่อให้ดูผลลัพธ์ง่ายขึ้น สรุปความเข้าใจได้ละเอียด กราฟแสดงผลอย่างละเอียด

รันโดย nginx กั้น frontend ทำ https ไปที่พอร์ต 3000
frontend ใช้ Next.js Tailwind ตามความเหมาะสมเพิ่มเติม ลดการใช้อิโมจิ และเลือกใช้ไอคอนแทน
backend ใช้ nodejs
database ใช้ postgres

ทั้งหมดรันด้วย docker ผ่าน compose สร้าง compose ให้ด้วย
